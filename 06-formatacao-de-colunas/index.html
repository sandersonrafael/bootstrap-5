É possível variar o ainhamento horizontal e vertical dos itens pelo pai ".row", usando align-items-center, justify-content-center e etc

Também é possível variar a posição de cada elemento individualmente usando o align-self-center, etc

É importante lembrar que todas essas classes também possuem modificadores de tamanhos de tela, como: align-items-xxl-center -> para telas grandes, o alinhamento é centralizado e nas menores não

Classes adicionais:

w-100 (e variações), determina o width de um elemento

order-last (e variações): determina a ordee que o elemento aparece

offset-1 (e variações) -> desloca uma coluna das 12 (ou mais, dep do número) como margem

ms-auto (e variações) -> cria uma margem automática (ou determinada) para a esquerda; ms vem de "margin start"

me-auto (e variações) -> similar ao ms, mas é para a direita; "marginend-auto"

mx-auto (e variações) -> é referente à margem no eixo x (tanto esquerda quanto direita)

comportamento da col sem elemento pai:

se ela não tiver um elemento pai como row, por exemplo, as divs seguem o padrão de ocupar sozinhas a linha toda, independente do tamanho do elemento

o número do col quem determina qual será o tamanho da div selecionada
