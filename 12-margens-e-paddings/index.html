Usamos as notações m-numero para representar as 4 margens < ^ v >, onde m-0 é 0rem, 1 é 0.25rem, 2 é 0.5rem, 3 é 1rem, 4 é 1.5rem e 5 é 3rem

Também é possível utilizar os paddings través das notações p-numero, com tamanhjos similares às margens m-numero

Para definir margens e paddings para a esquerda, utilizamos ms-n ou ps-n (s de start)

Para a direita, usamos o e: me-n && pe-n (e de end)

Para o topo, usamos o t: mt-n && pt-n (t de top)

Para o fundo, usamos o b: mb-n && pb-n (b de bottom)

Para o eixo x (esquerda e direita), usamos o mx-n e px-n

Para o eixo y (cima e baixo), usamos o my-n e py-n

Para margens ou paddings automáticos: m-auto, p-auto, mx-auto... etc

Todas essas variações, possuem as adequações para responsividade, como mx-sm-5, etc
