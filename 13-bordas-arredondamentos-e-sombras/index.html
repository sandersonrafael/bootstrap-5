Utilizando bordas

Para adicionar bordas ao elemento, utilizamos a classe border

Para adicionar cores, utilizamos border-tipoDaCor. Ex: border-dark

Para definir o tamanho, usamos o border-numeroDoTamanho

Para controlar a opacidade da borda, utilizamos o border-opacity-numero. Ex: border-opacity-75

É possível usar bordar para determinadas direções: border-(start | end | top | bottom)

Além disto, podemos usar tamanhos específicos, como border-start-5 ou border-5 border-start (as duas maneiras resultam no mesmo)

No bootstrap 5, não é possível adicionar tamanhos específicos com responsividade para as bordas (variando com o tamanho da tela)

Arredondamentos

Usamos o rounded e suas variações... rounded-top, rounded-start, rounded-5, etc

Para um elevado arredondamento, usamos o rounded-pill

Para um máximo arredondamento, usamos o rounded-circle

Sombra

padrão: shadow-none -> sem sombras

shadow-sm -> sombra quase imperceptível | shadow -> sombra média | shadow-lg -> sombra maior
