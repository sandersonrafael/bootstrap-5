Alinhamento

Usamos os text-posicaoDoAlinhamento. Como text-start -> alinhamento à esquerda

Também se aplicam os modificadores de responsividade, como text-sm-center

Quebra de linhas

text-nowrap para não haver quebra de linha

text-break para adicionar quebra de palavras longas (útil para colunas de tabelas estreitas, por exemplo)

Maiúsculas e minúsculas

text-lowercase -> todas as letras minúsculas

text-uppercase -> todas as letras maiúsculas

text-captalize -> todas as palavras com iniciais maiúsculas

Tamanhos de fontes

Usamos as variações do fs-numero, onde fs-1 é a maior e fs-6 a menor

Altura da linha de texto

Utilizamos as variações de lh-(1 | base | lg | sm) para modificar a altura da linha, que representam respectivamente: altura mínima, altura normal, altura longa e altura "dupla" das linhas

Fontes com todas letras monoespaçadas: font-monospace

link

Para links (a), podemos usar a classe text-reset -> o link assume as características da fonte do elemento pai

Para remover o sublinhado, usamos a classe text-decoration-none

Para o texto com aspecto riscado -> text-decoration-line-throug

Caso deseje explicitar o sublinhado, usamos o text-decoration-underline

Além disso, é possível truncar o texto, mostrando apenas uma linha e reticências, com o text-truncate
