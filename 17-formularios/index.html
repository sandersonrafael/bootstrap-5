Classes form

As classes são em geral: form-tipoDoElemento

Para o input, a classe é form-control

Para textos informativos a classe é a form-text

A div que contém o checkbox quem usa a classe form-check, os demais relacionados ao check são: input-check-algumacoisa

Para mudar o visual do botão, usamos a classe btn e btn-cor

Para desabilitar um campo, a classe disabled é usada

readonly permite copiar e colar o texto dentro do input, diferente do disabled que não permite

Tamanho -> para variar o tamanho do campo, podemos usar o form-control-lg ou sm

form-control-paintext -> o valor do campo fica como um texto comum

input:file -> utilizamos o form-control também para dar aparência do bootstrap

input:color -> igualmente input-control
