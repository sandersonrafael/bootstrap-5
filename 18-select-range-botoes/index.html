Select

form-select -> dá o estilo do bootstrap ao select de um formulário. Essa classe faz com que ocupe 100% do espaço disponível

Para delimitar o espaço, podemos englobar o select em uma div com col-n

Para variar o tamanho, há o form-select-sm e o form-select-lg

Usando o atributo multiple, o select fica automaticamente aberto e com barra de rolagem. O atributo size="n" define quantas linhas ficarão expostas

Além disso, dá para usar o atributo de disabled

Input range

classe form-range, e é necessário definir um min, max, step e value (padrão)

Botões

As mesmas estilizações dos botões podem ser usados para links

Utilizamos a classe btn para indicar que representa um botão e btn-tipoDeCor para definir a cor. Ex: class="btn btn-success"

Para botões secundários, por exemplo, há a classe btn-outline-tipoDaCor -> botão branco com borda colorida e quando passa o mouse, fica colorido

Para modificar o tamanho, temos o btn-lg e o btn-sm
