Títulos flutuantes

São os títulos que ficam centralizados no input, até o usuário clicar. Quando clica, ele vai para cima do input

O elemento pai precisa ter a classe form-floating, o input precisa ter um placeholder vazio e ter a classe form-control

É necessário haver um label para o input com innerText e com classe form-label

Também é possível ter título flutuante em ‹textarea›

Assim como em select. Porém, o select não deixa o label no meio do espaço como nos inputs
