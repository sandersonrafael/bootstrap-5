Múltiplos campos de formulário por linha

Para controlar o espaçamento entre inputs, utilizamos o gap, ou g-numero. Ex: g-3 -> espaçamento de 1rem

Para definir o tamanho que o label vai ocupar, usamos a classe "col-form-label col-n" e a classe col-n na div do input

Para variar a altura, usamos o col-form-label-sm ou col-form-label-lg no label
