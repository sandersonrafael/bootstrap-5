Formatação de tabelas

Podemos usar a classe table no elemento table. Isso já vai dar um visual mais interessante

Podemos variar a cor da table com table-cor -> ex: table-primary, secondary, danger, etc

É possível adicionar cor somente a algum elemento da tabela, como o thead

Podemos fazer a tabela ter cor zebrada, com o striped: table-striped ou table-striped-columns

Já o table-hover, faz com que a tabela tenha variação de cor com o hover do usuário

Para destacar alguma linha ou coluna, posso usar a classe table-active

Para adicionar bordas em todos os cantos, há a classe table-bordered; para remover: table-borderless

Para exibir a tabela de maneira mais compacta: table-sm

Adicionar borda de destaque para uma linha, tbody, thead ou tfoot: table-group-divider

Para alinhar verticalmente os elementos da tabela, podemos usar o align-middle

Para posicionar o caption em outro lugar, podemos usar o caption-local. Ex: caption-top
