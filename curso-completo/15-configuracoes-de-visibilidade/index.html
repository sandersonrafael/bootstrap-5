Propriedade display

d-block -> display do tipo bloco, sem elementos ao lado

d-inline -> display do tipo em linha, sem width e height

d-inline-block -> elemento do tipo bloco em linha, com propriedade width e height

d-none -> o elemento existe na estrutura da página, mas não é exibido.

Importate -> para não exibir em telas maiores, usamos o d-sm-none, d-md-none, etc. Já para não exibir em telas menores, usamos: d-none d-md-block, etc

Há também o d-grid, d-flex, d-line-flex e as suas variações com tamanhos de tela, d-sm-flex, por ex

Exemplo

Para que ao tentar imprimir, o elemento não seja parte da impressão, utilizamos o d-print-none

Para que o elemento só seja mostrado caso usuário tente imprimir e não no uso normal do site: d-none d-print-block

Outro exemplo

Classe invisble -> elemento não é mostrado, mas ocupa espaço na página

Transbordo de conteúdo (overflow)

overflow-hidden -> esconde o conteúdo que transborda

overflow-auto -> quando há transbordo de conteúdo, abre barra de rolagem

overflow-visible -> comportamento padrão, que mostra caso passe

overflow-scroll -> sempre com barras visíveis
