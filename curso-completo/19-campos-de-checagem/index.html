Campos de checagem padrões

São representados pelos checkbox e o radio

O elemento pai do checkbox precisa da classe form-check e o elemento checkbox precisa da classe form-check-input

Para os elemnetos ficarem lado a lado, utilizamos a classe form-check-inline no elemento pai

Para alinhas os checkbox à direita, utilizamos o form-check-reverse no elemento pai

checkbox switch

utiliza-se a classe form-switch no elemento pai do input checkbox

checkbox do tipo botão

Para usar um botão como checkbox (que fica ativado ou desativado), usamos a classe:

btn-check no input checkbox e btn btn-outline-tipoDeCor

Isso também se aplica a inputs do tipo radio
