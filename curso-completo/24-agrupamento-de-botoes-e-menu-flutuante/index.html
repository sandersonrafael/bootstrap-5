Botões agrupados

No elemento pai dos botões, adicionar a classe btn-group

Para botões checáveis, utilizamos a classe btn-check em inputs do tipo radio ou checkbox

Para botões que possuem menus suspensos (dropdown), adicionamos a classe dropdown no elemento pai;

Após isso, é necessário adicionar um botão com a classe dropdown-toggle, com o atributo data-bs-toggle="dropdown"

Ao lado do botão, um elemento irmão ul deve ser adicionado, cujos lis possuem links (a) com classe dropdown-item
