Alertas

Para adicionar alertas, utilizamos a classe alert alert-tipoDaCor

Para adicionar links ao alert, a classe link-alert deve ser utilizada numa tag do tipo link

Para adicionar título -> alert-heading

Adicionar botão para fechar o alerta -> alert-dismissible na classe pai do alerta; Adicionar um botão com a classe btn-close e atributo data-bs-dismiss="alert"; Para adicionar suavidade ao fechar o alerta: adicionar as classes fade show na div pai do alerta

Exemplo de uso:

Lista agrupadas (ou lista de agrupamentos)

Criar uma div com a classe list-group

Os elementos filhos (li) devem conter a classe list-group-item

Para deixar com aparência destacada, utilizar a classe active

Para destaque "negativo", utilisar disabled

Para links clicáveis, utilizar itens com tag >a> e com classes list-group-item list-group-item-action

Para botões clicáveis, utilizar itens com tag >button> e com classes list-group-item list-group-item-action

Para retirar as bordas ao redor, podemos utilizar a classe list-group-flush na classe pai do list-group

Para list-groups numerados, utilizamos na div pai as classes list-group list-group-numbered
